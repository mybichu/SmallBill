# SmallBill
一个java版的小小记账本程序

# 主要功能
1. 消费一览
2. 记录消费信息
3. 消费分类管理
4. 月度消费柱状图显示
5. 设置本月预算和MySQL数据库路径
6. 备份
7. 恢复

# 环境
*  OS:win10-64bit
*  IDE: eclispe
*  database:mysql-5.7
